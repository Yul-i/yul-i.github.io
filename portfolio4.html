MomCar

Mobile Application

HTML5, CSS3, JAVASCRIPT, JQUERY

아픈 아이 부모를 위한 호출 서비스 &건강 기록을 할 수 있는 헬스 케어 서비스(가상)

Keywords /
  • 육아/헬스케어/의료/응급상황/라이프스타일/호출서비스
Problems /
  • 아이와 엄마가 급하게 병원을 가야하는데 전문적인 호출 서비스가 존재하지 않는다.
  • 위급 상황시 신속하고 안전하게 병원으로 가는 이동수단이 필요하다.
  • 아이의 성장과 건강 관련하여 기록할 수 있는 곳이 필요하다.
  • 육아에 필요한 기초 의료지식이 보기 편하게 모아져 있었으면 좋겠다.
  • 복잡하지 않고 단순한 UI가 사용하기에 편리하다.
Solutions /
  • 호출 서비스 : 차량 호출
  • 내 위치 인근 병원 정보 제공 : 병원 선택 필터, 과목별
  • 아이 건강 관리 : 정보 등록, 기록형(일자별)
  • 진료 내역 기록(체온, 복용한 해열제, 약품, 증상 기록 사용자 등록)
  • 진료 기록 내역 보기 : 히스토리 나열(일자별, 최신순)
  • 응급 상황, 병원 정보 제공 : 야간, 공휴일 운영 정보
  • 건강 정보 제공
